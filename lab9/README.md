# hashlab (Go)

Лабораторная: оценка качества хеш-функций на случайных ключах длиной 6.

## Что делает программа

Генерирует N случайных строк (0-9, A-Z, a-z), преобразует ключ в число двумя способами:
- additive: сумма ASCII-кодов символов
- xor: sum += (ASCII(key[i]) XOR r[i]), где r — фиксированный массив из 6 байт

Далее вычисляет адрес (0..M-1) двумя методами:
- division: h(k) = k mod M
- multiplication: h(k) = floor(M * frac(k*A)), A = (sqrt(5)-1)/2

Итого 4 комбинации. Для каждой:
- считает сколько ключей попало в каждую ячейку
- считает коллизии на ячейку (max(0, keys-1))
- пишет CSV (удобно для Excel)
- пишет summary.csv с итоговыми метриками.

## Сборка и запуск

```bash
go run ./cmd/hashlab -m 101 -n 1000 -out out
```

Параметры:
- `-m` размер таблицы M
- `-n` количество ключей N
- `-seed` фиксированный seed (0 = текущее время)
- `-out` папка вывода
- `-sep` разделитель CSV (по умолчанию `;`)

После запуска в `out/` будут файлы:
- additive_division.csv
- additive_multiplication.csv
- xor_division.csv
- xor_multiplication.csv
- summary.csv
